    ralsei.wrappers.pop_id_fields(*id_fields: str, keep: bool = False) → collections.abc.Callable[[ralsei.wrappers.OneToMany], ralsei.wrappers.OneToMany]¶

    Create function wrapper that ‘pops’ id_fields off the keyword arguments, calls the inner function without them, then re-inserts them into the output rows

    >>> @pop_id_fields("id")
... def foo(a: int):
...     yield {"b": a * 2}
...
>>> next(foo(id=5, a=3))
{"id": 5, "b": 6}

    Additionally, popped field names are saved into the function’s metadata, so that tasks can use them for inferring IdColumns

    Parameters:¶
    *id_fields: str¶

    keyword arguments to pop

    keep: bool = False¶

    if True, the popped arguments would still be passed to the inner function

    >>> @pop_id_fields("year")
... def foo(year: int, name: str):
...     yield {"html": download(year, name) }
...
>>> next(foo(year=2015, name="Tokyo"))
{"year": 2015, "json": {...}}
